# Visualizations

Visualizations in this project leverage matplotlib and seaborn.

## Key Visuals Generated:
1. **Average Power by Cloud Cover (Bar Chart):** Shows how binned cloud cover categories (Low, Medium, High) affect power.
2. **Solar Radiation vs Power (Line Trend):** Plots sorted radiation against power to show the linear-like growth.
3. **Temperature Distribution (Box Plot):** Demonstrates the variance of power at different temperature levels.
4. **Radiation vs Power (Scatter):** Provides a granular look at the clustering of data points.
